# toolbox
_A container of handy tools_

## Installation
Add toolbox to PATH in `~/.bash_profile`
```
export PATH="$PATH:/path/to/toolbox/bin"
```

## Commands
build toolbox
```
tb build
```
use a tool
```
tb http github.com
```
list tools installed (excluding os packages)
```
tb list
```
enter toolbox as interactive bash session
```
tb enter
```
